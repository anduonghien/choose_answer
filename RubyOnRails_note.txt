8/ Ket noi voi db co san
1. modify the database.yml to point to the database you want to generate from
2. rake db:schema:dump, now schema.rb should have scripts to create all the tables
3. modify the database.yml to point back to the correct database
4. rake db:schema:load, now the destination database should have all the tables
